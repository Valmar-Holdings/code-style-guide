  • ‚úã View routes should have no namespace.
  • ‚úã View routes should use resource controllers.
  • ‚úã View route controllers should be named after the blade view folder they are responsible for.
  • ‚úã View route controller methods should implement the restful naming scheme.
  • ‚úã View route controllers should only be responsible for a single model, but responsible for all views that pertain to that model, for example:
    • /resources/views/reports/index.blade.php
    • /resources/views/reports/create.blade.php
    • /resources/views/reports/show.blade.php
    • etc...
